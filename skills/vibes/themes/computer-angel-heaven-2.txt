THEME: computer-angel-heaven-2
NAME: Computer Angel Heaven
MOOD: Luminous, circuitry-divine, pearlescent, tech-ethereal

DESCRIPTION:
A dashboard interface where sacred geometry meets printed circuit boards. A wide
hero banner shows angel wings traced in PCB copper lines over a frosted sky, with
a sidebar of icon nodes connected by faint trace paths. Below, a responsive card
grid holds glassmorphic panels for data, charts, feeds, and chat — each card
decorated with hairline circuit traces at its corners. The palette blends icy
pearl backgrounds with warm gold conductor accents and soft lavender highlights.
Feels like the control panel of a benevolent AI in the clouds.

BEST FOR:
  - Dashboard and analytics apps with card-based layouts
  - Server monitoring, system status, infrastructure panels
  - Collaborative project trackers, kanban boards
  - AI tool interfaces, model playgrounds, prompt builders
  - Network/topology visualizers, connection maps
  - IoT device managers, smart-home panels
  - Any app mixing technical data with a calm, aspirational aesthetic

NOT FOR:
  - Dense spreadsheet-style apps (glass blur + many cells = perf cost)
  - Dark-mode-first apps (fundamentally a light-sky palette)
  - Brutalist or raw-industrial aesthetics (this is polished and soft)
  - High-contrast accessibility-critical apps (low-contrast pastels)
  - Text-heavy reading apps — novels, long articles (too decorative)

ADAPTATION NOTES:
  - If app has TABLES: Frosted-glass table container with generous row height
    (52px+). Header row in slightly darker glass. Gold left-border on hover
    row. Circuit-trace corner decorations on the table card. Max 5-6 columns
    to keep the airy feel.
  - If app has CHARTS: Place inside hero-sized cards. Use gold for primary
    series, sky-blue for secondary, lavender for tertiary. Transparent chart
    background so the page gradient shows through. Axis text in muted blue-gray.
  - If app has FORMS: Inputs get frosted backgrounds with 1px translucent
    borders. Focus adds gold glow ring. Labels in small-caps muted text.
    Submit button is gold pill with white text. Group inputs in glass cards.
  - If app has NO IMAGES: Circuit-trace SVG decorations and subtle radial
    gradients inside cards carry the visual interest. Add a faint halo glow
    behind key headings.
  - If app has TABS/FILTERS: Frosted pill tab bar above the card grid. Active
    tab gets gold bottom-border and brighter glass fill. Inactive tabs are
    transparent with muted text.
  - If app has SIDEBAR: Use the icon-node pattern — a narrow vertical glass
    strip with round icon buttons connected by faint vertical trace lines.
    Active icon gets a gold halo glow behind it.
  - If app has CHAT/MESSAGES: Sent messages in gold-tinted glass bubbles,
    received in blue-tinted glass. Timestamp in lavender micro-text. Message
    input at bottom with glass background and gold send button.
  - If app has REAL-TIME DATA: Use the progress-bar pattern from reference.
    Gradient fill from sky-blue to gold. Pulse animation on live indicators.
    "Live" badge is a green dot + label in a frosted pill.

DESIGN PRINCIPLES:
  Typography:
    - Headings: Light weight (300), wide tracking (0.03em+). Geometric sans
      like Inter, Outfit, or Sora. Airy and expansive — never heavy.
    - Body: Regular weight (400), line-height 1.65. Comfortable reading.
    - Labels/metadata: Uppercase micro-text (0.7rem), 500 weight, 0.08em spacing.
    - Stats/numbers: Tabular figures, 1.2rem+, gold accent color for key values.
    - AVOID: Bold/black weights, monospace for UI labels, condensed faces.

  Spacing:
    - Generous whitespace. Card padding: 24-32px. Grid gap: 20-24px.
    - Section spacing: 32-48px. Elements float in space, not packed.
    - Sidebar column: 72-80px wide with vertically centered icon nodes.
    - Hero card spans full width, min-height 200px.

  Borders:
    - Translucent 1px borders on glass surfaces (--comp-border).
    - No hard opaque lines. Everything blends into the sky.
    - Active/focus elements use --comp-border-accent (gold tint).
    - Border-radius always generous (--comp-radius or larger).

  Shadows and Depth:
    - Layered box-shadows:
      `0 4px 24px var(--comp-shadow), 0 1px 4px var(--comp-shadow)`
    - Hover adds glow: `0 0 24px var(--comp-glow)`
    - Glass surfaces: `backdrop-filter: blur(22px) saturate(1.15)`
    - Cards float above the background — visible elevation change on hover.

  Background:
    - Page: multi-stop gradient — warm pearl → pale sky-blue → subtle lavender.
    - Optional: faint circuit-board trace pattern as SVG background overlay.
    - Feeling: open sky with hairline technological grid beneath.

PERSONALITY:
  This theme speaks like a benevolent machine intelligence — precise yet warm,
  technical yet reverent. It knows the data but presents it with grace. Empty
  states: "Awaiting signal — ready when you are." Errors: "A connection
  faltered. Attempting graceful recovery." Buttons use calm imperatives:
  "Initiate," "Ascend," "Transmit," "Illuminate." Status labels: "Online,"
  "Synced," "Ascending." The voice is the calm narration of a system that
  cares about its users — never cold, never frantic.

ANIMATIONS:
  Tempo: Measured and fluid. Nothing mechanical. Everything eases gracefully.

  - Page load: Cards float in from below (translateY 16px → 0, opacity 0 → 1,
    650ms ease-out, staggered 90ms per card).
  - Hover on cards: Subtle lift (translateY -4px) + gold glow intensifies +
    border brightens. 300ms cubic-bezier(0.22, 1, 0.36, 1).
  - Circuit traces: Hair-thin gold lines at card corners that extend on hover
    (width: 0 → 30px, 400ms ease-out). Creates a "powering up" feel.
  - Button hover: Gold glow expands (box-shadow scales outward), background
    brightens a step. 250ms ease.
  - Active/click: Scale(0.97) spring-back. 150ms ease.
  - Modal/dialog: Fade + slight scale (0.96 → 1.0) with backdrop blur.
    400ms ease-out.
  - Sidebar icon hover: Gold halo fades in behind icon (radial-gradient
    pseudo-element). 300ms ease.
  - Sparkle pulse: 4-point star particles at 0.3-0.6 opacity, scale 0.8 →
    1.2, 3-5s infinite. Scattered sparingly across hero area.
  - Progress bars: Gradient fill animates left-to-right on load (800ms ease).
  - Live indicators: Green dot with slow pulse (opacity 0.6-1.0, 2s infinite).
  - Use `cubic-bezier(0.22, 1, 0.36, 1)` for the signature floaty ease.
  - AVOID: Bounce, elastic, mechanical snap. Everything drifts like clouds.

SVG ELEMENTS:
  Decorative SVGs merge circuit-board motifs with celestial symbolism.

  - Circuit-trace corners: L-shaped PCB traces at card corners, drawn in gold.
    Thin strokes (1-1.5px), rounded line-caps, opacity 0.2-0.35.
    ```svg
    <svg viewBox="0 0 40 40" class="trace-corner trace-tl">
      <path d="M0 20 L0 4 Q0 0 4 0 L20 0" fill="none"
        stroke="var(--comp-trace)" stroke-width="1.5"
        stroke-linecap="round" />
      <circle cx="20" cy="0" r="2" fill="var(--comp-trace)" />
    </svg>
    ```

  - Halo ring: Thin gold ellipse behind headings or avatars. Animate with
    stroke-dasharray draw-in on page load.
    ```svg
    <svg viewBox="0 0 100 40" class="halo-ring">
      <ellipse cx="50" cy="20" rx="45" ry="14"
        fill="none" stroke="var(--comp-accent)" stroke-width="1.2"
        opacity="0.4" stroke-dasharray="300"
        stroke-dashoffset="300" />
    </svg>
    ```

  - Chip nodes: Small square-with-pins icons at sidebar endpoints or as
    decorative markers. 16-20px, gold stroke, no fill.

  - Angel wing accents: Simplified bilateral wing curves flanking the hero
    heading or logo. Low opacity (0.12-0.20) so they read as background
    texture. Gold or sky-blue stroke.

  - Sparkle stars: 4-point stars (4-8px) scattered via absolute-positioned
    elements. Slow pulse animation. Max 5-8 per visible viewport.
    ```svg
    <svg viewBox="0 0 20 20" class="sparkle-star">
      <path d="M10 0 L12 8 L20 10 L12 12 L10 20 L8 12 L0 10 L8 8 Z"
        fill="var(--comp-sparkle)" />
    </svg>
    ```

  - PCB background grid (optional): Ultra-faint straight-line grid with
    occasional right-angle turns, overlaid on the page background at
    opacity 0.04-0.06. Gives a subliminal technological substrate.
